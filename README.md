## Project Setup

Step 1

Node.js install kar lena

Step 2

Repository clone kar lena

```bash
git clone https://github.com/vhatkarajinkya07-creator/HamRahi
```

Step 3

Frontend directory me jao

```bash
cd frontend
```

Step 4

Dependencies install karo

```bash
npm install
```

Step 5

Development server start karo

```bash
npm run dev
```

Step 6

Browser me open karo

```text
http://localhost:5173
```
